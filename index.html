Welcome to the pinybot repository homepage

What is this?

We know that Tinychat bots have existed for a long time. Started mainly with smaller individuals, namely MegaLoler and re-constructed by Nola (GitHub at the bottom). This is a redefined version for people who intend to use bots on the rooms, however we have not tried to be too specific with the style of coding and the commands as we would like YOU to improve it!

The bot will be updated by all of us as time progresses and whatever new Tinychat changes are made, we will have to adhere to them, to keep this bot functional. Just briefly we would like to mention pinylib author nortxort for the base of this bot and for his major contributions with creating it. As well as that a special mention to Nola who made the pinychat library which was the clearest python library implementation of Tinychat (redefined from MegaLoler's work). Without that we could have never made any of this.

The repository

Our repository is located at oddballz/pinybot (a simple group of members who all share the same interest). Feel free to browse the pinybot repository and view the README before you install any dependencies you may or may not require. The pinybot-OpenShift branch of the repository contains our sample server files for OpenShift setup and a nearly "click to install and use" configruation.

If you already have an OpenShift account setup, be sure to just click on the logo below to install straight to OpenShift and deploy!
