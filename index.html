Welcome to the pinybot repository homepage

What is this?

We know that Tinychat bots have existed for a long time. Initially started by smaller individuals, the likes of MegaLoler and re-constructed for Python by Nola (GitHub below). This is a redefined version for people who intend to use bots in rooms, however we have not tried to be too specific with the style of coding and the commands, as we would like YOU to improve it further!

The bot will be updated by all of us as time progresses and whatever new changes Tinychat makes, we will match side-by-side to provide the most functional and easy to use bot. Just briefly we would like to mention pinylib author nortxort for the base of this bot and for his major contributions with creating it.

As well as that a special mention to Nola who made the pinychat library, which was the clearest python library implementation of Tinychat (redefined from MegaLoler's work). Without that we could have never made any of this.
